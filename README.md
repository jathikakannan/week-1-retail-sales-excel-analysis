Week 1 Retail Sales Excel Analysis

Objective

This project focuses on understanding and analyzing a retail sales dataset using Microsoft Excel. The goal was to learn data cleaning, formatting, analysis, sorting, filtering, and visualization techniques.

Dataset

Retail Sales Dataset

Tasks Completed

* Dataset Understanding
* Data Cleaning and Formatting
* Data Quality Check
* Basic Analysis
* Sorting and Filtering
* Charts Creation
* Observations and Insights
 Key Findings

* Highest Sales Category: Electronics (₹156,905)
* Lowest Sales Category: Beauty (₹143,515)
* Female Customer Sales: ₹232,840
* Male Customer Sales: ₹223,160
* Total Quantity Sold: 2,514 Units
* Average Customer Age: 41.39 Years
